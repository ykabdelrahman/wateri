## Watri
A Flutter app that reminds users to drink water with customizable intervals and persistent tracking.

## 🚀 Getting Started

To run this project on your local machine:

```bash
# Clone the repo
git clone https://github.com/ykabdelrahman/watri.git
cd your-repo-name

# Get dependencies
flutter pub get

# Run the app
flutter run
```
